# CLEAR VISION

Clear Vision is a premium eyewear online store offering eyeglasses, sunglasses, computer glasses, kids glasses, contact lenses and frames — all from a single place. Built with a bold dark theme, luxury UI, and a full admin dashboard.

---

## Features

- Browse & filter eyewear by category, shape, color and size
- Product detail page with lens selector and add to cart
- Cart, checkout and payment flow
- My Orders page with order tracking and cancel order
- Virtual Try-On (coming soon) and Home Trial
- Free Eye Test info at partner stores
- Admin dashboard — manage products, orders, users
- Add products with category (Eyeglasses, Sunglasses, Contact Lenses, Frames)
- Dark premium UI throughout — Bebas Neue, Cormorant Garamond, DM Sans

## Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/ayaanansari8/ClearVision-Optics-Ayaan.git

# Navigate into the project
cd ClearVision-Optics-Ayaan

# Run with Docker
docker-compose up --build
```

App runs on `http://localhost:3001`
